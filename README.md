# My .minecraft.fabric folder for Hypixel Skyblock in modern versions

This is an hugely optimized setup so expect to get very high FPS, but you must use the latest version of Java (e.g 26)

Make sure to combine this with my launcher wrapper and/or the [JVM args](https://github.com/TheDGOfficial/minecraft-launcher-launcher?tab=readme-ov-file#recommended-jvm-flags) for best results

Most mods are already configured, but the modpack might lack some mods for performance reasons, although the lacking mods are mainly optional stuff, the modpack still has all the essential stuff like solvers, maps, waypoints, etc.
